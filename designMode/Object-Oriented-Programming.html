  1. 封装

    通过 this 定义的属性或者方法是该对象自身拥有的, 所以每创建一个新对象, this 指向的属性和方法都会得到相应的创建

    而通过 prototype 继承的属性和方法是每个对象通过 prototype 访问得到, 所以每次通过创建类创建一个新对象, 这些属性和方法不会新建

    Javascript 的 函数作用域, 声明在函数内部的变量以及方法在外界访问不到

    通过 this 创建的方法, 可以访问对象的共有属性 / 方法, 也可以访问类(创建时)或对象自身的私有属性 / 方法 – 特权

    调用特权方法 – 构造器
